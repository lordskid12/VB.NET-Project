 🏥 Integrated Management System (VB.NET)
A Windows Forms application developed using **VB.NET** to automate Integrated Hospital Billing and Student Loan Management System.

 Project Overview
This system replaces manual calculations and reduces errors by providing automated billing and loan processing with a clean user interface.

🏥 Hospital Billing Module
Features:
* Enter patient name
* Select ward type:
  * General Ward (200/day)
  * Semi-Private (350/day)
  * Private (500/day)
* Enter number of days
* Select optional services:
  * Lab (150)
  * X-Ray (100)
  * Surgery (1000)

 System Calculates:
* Ward Charges
* Treatment Charges
* VAT (12%)
* Final Bill
✔ Displays **Premium Billing Category** if bill > 5000
✔ Print professional receipt 

🎓 Student Loan Module
Features:
* Input student details:
  * Name
  * GPA (0.1 – 4.0)
  * Family Income
  * Loan Amount
* Check eligibility
* Calculate repayment

Outputs:
* Monthly Payment
* Total Payment
* Printable receipt

🧾 Receipt Features
* Professional invoice design
* Centered layout
* Borders and sections
* Date & Time included
* Hospital logo
